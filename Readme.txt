== Description ==

A simple widget that uses the native WordPress media manager to add images to widget of your site. You can add any number of images. You can give Links, Title, Descriptions for each images .

== Installation ==

1. Upload `image-in-the-widget` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Drag Image In The Widget to your desired sidebar.
4. Upload your image and you can give desired options there.
